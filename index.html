Guia rápido para Git/Github

Link do curso.

  • Iniciar um repositório: 
    $ git init
  • Verificar o staus do repositório: 
    $git status
  • Adicionar arquivos ao controle de versão: 
    $ git add "nome_do_arquivo
    ou para todos: 
    $ git add *
    ou 
    $ git add .
  • Criar versões do código com commit 
    $ git commit -m "commit inicial"
    Caso seja necessário, o Git vai pedir pra vc configurar seus dados:
    • Configurar e-mail: 
      $ git commit --global user.email "seuemail@provedor.com"
    • Configurar seu Usernam: 
      $ git config --global user.name "seuUserName"
